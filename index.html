Voici comment déployer ton site sur GitHub Pages — c'est gratuit et tu obtiens une URL publique type ton-nom.github.io/zemo-rack.

Étape 1 — Crée un compte GitHub Va sur github.com et crée un compte si tu n'en as pas.

Étape 2 — Crée un nouveau repository Clique sur "New repository", donne-lui un nom (ex: zemo-rack), laisse-le en Public, puis clique "Create repository".

Étape 3 — Upload ton fichier Sur la page du repo, clique "uploading an existing file", glisse-dépose ton fichier zemo-rack-investor-site.html, puis renomme-le index.html avant de valider. Clique "Commit changes".

Étape 5 — Récupère ton URL Après 1-2 minutes, ton site est live à : https://ton-username.github.io/zemo-rack

GitHub Pages prend parfois 2-3 minutes pour la première mise en ligne. Pour mettre à jour le site plus tard, tu ré-uploades juste un nouveau index.html au même endroit.
